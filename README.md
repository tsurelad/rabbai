# Rabbai — רב אישי

> An AI-assisted personal *rabbi* (rav) that studies classical Jewish sources with you in Hebrew, grounded in a local corpus of Tanakh, Mishnah, Talmud Bavli & Yerushalmi, Mishneh Torah, Shulchan Aruch, Midrash, Halakha, Chasidut, Machshavah & Mussar, Kabbalah, Tefillah and Responsa — all sourced from [Hebrew Wikisource](https://he.wikisource.org) (CC BY-SA 4.0).
>
> English speakers: the project itself and all source texts are in Hebrew. See below.

## מה זה?

**Rabbai** הוא פרויקט של "רב אישי" מבוסס בינה מלאכותית, הלומד איתך תורה בעברית מתוך מאגר מקומי של טקסטים יהודיים קלאסיים — תנ"ך, משנה, תלמוד בבלי וירושלמי, משנה תורה לרמב"ם, שולחן ערוך, מדרש, הלכה, חסידות, מחשבה ומוסר, קבלה, תפילה, ושו"ת — כולם שאובים מ-[ויקיטקסט העברי](https://he.wikisource.org) תחת רישיון Creative Commons BY-SA 4.0.

הסוכן נועד לסייע בלימוד, בבירור מקורות, ובהבנת מחלוקות ראשונים ואחרונים — ולא להחליף רב בשר ודם לפסיקת הלכה למעשה.

## איך זה עובד

1. **הנחיית הסוכן** נמצאת ב-[`AGENTS.md`](AGENTS.md) — כולל טון, מבנה תשובה, ונוהל הפעלת סוכני משנה. [`CLAUDE.md`](CLAUDE.md) ו-[`GEMINI.md`](GEMINI.md) הם קישורים סימבוליים לאותו קובץ, כדי שסוכנים שונים (Claude, Gemini, ועוד) יטענו את אותן ההנחיות.
2. **מפת הידע** נמצאת ב-[`summaries/INDEX.md`](summaries/INDEX.md) — שער הכניסה לשבעת מאגרי העומק: מקרא, משנה, בבלי, ירושלמי, משנה תורה, שולחן ערוך, ותרי"ג מצוות. זה נקודת הפתיחה של הסוכן לכל שאלה תורנית מהותית.
3. **המקורות עצמם** יושבים תחת [`source/`](source/), מאורגנים לפי מדפי ארון הספרים היהודי.

כל ציטוט חייב להישאב מ-`source/` — הסיכומים ב-`summaries/` הם מפת דרכים בלבד, לא מקור ציטוט.

## מבנה הפרויקט

```
.
├── AGENTS.md               # הנחיית הסוכן (קאנוני)
├── CLAUDE.md -> AGENTS.md  # קישור סימבולי — Claude Code
├── GEMINI.md -> AGENTS.md  # קישור סימבולי — Gemini
├── summaries/              # מפת דרכים: INDEX.md + שבעה סיכומי עומק
└── source/                 # מאגר הטקסטים המלא (ויקיטקסט העברי)
    ├── מקרא/
    ├── מקרא - טעמים/
    ├── מקרא מבואר/
    ├── מקרא ותרגום ורש"י/
    ├── פרשני המקרא/
    ├── משנה/
    ├── משנה מבוארת/
    ├── תלמוד בבלי/
    ├── תלמוד ירושלמי/
    ├── חיבורים על הש"ס/
    ├── ברייתות/
    ├── משנה תורה לרמב"ם/
    ├── שולחן ערוך/
    ├── הלכה/
    ├── מדרש/
    ├── חסידות/
```

לפירוט מלא של מבנה הקבצים, מוסכמות השמות, ואופן האיתור המהיר של פסוק, דף או סימן — ראה את [`AGENTS.md`](AGENTS.md).

## שימוש

הפרויקט מיועד לעבוד עם סוכני בינה מלאכותית שמכבדים את קובץ ההנחיות (`AGENTS.md` / `CLAUDE.md` / `GEMINI.md`). פתיחת הפרויקט עם סוכן כמו Claude Code או Gemini CLI תטען את ההנחיות אוטומטית, והסוכן יתפקד כרב אישי לפי הפרוטוקול המתואר שם.

## סייג הלכתי

הסוכן עוזר ללימוד ולבירור, ואינו מחליף רב בשר ודם לשאלות הלכתיות חמורות (כגון גיטין, הוראת טריפה, דיני עירוב, יום הכיפורים וכו'). לפסיקה מעשית — שאל רב בקהילתך.
