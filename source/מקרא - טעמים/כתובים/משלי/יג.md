# משלי פרק יג (עם טעמים)

**(א)** בֵּ֣ן מ֣וּסַר אָ֑בוְ֝לֵ֗ץ לֹא־שָׁמַ֥ע גְּעָרָֽה׃
 **(ב)** מִפְּרִ֣י פִי־אִ֭ישׁ יֹ֣אכַל ט֑וֹבוְנֶ֖פֶשׁ בֹּגְדִ֣ים חָמָֽס׃
 **(ג)** נֹצֵ֣ר פִּ֭יו שֹׁמֵ֣ר נַפְשׁ֑וֹפֹּשֵׂ֥ק שְׂ֝פָתָ֗יו מְחִתָּה־לֽוֹ׃
 **(ד)** מִתְאַוָּ֣ה נַפְשׁ֣וֹ עָצֵ֑לוְנֶ֖פֶשׁ חָרֻצִ֣ים תְּדֻשָּֽׁן׃
 **(ה)** דְּבַר־שֶׁ֭קֶר יִשְׂנָ֣א צַדִּ֑יקוְ֝רָשָׁ֗ע יַבְאִ֥ישׁ וְיַחְפִּֽיר׃
 **(ו)** תִּצֹּ֣ר תׇּם־דָּ֑רֶךְוְ֝רִשְׁעָ֗ה תְּסַלֵּ֥ף חַטָּֽאת׃
 **(ז)** יֵ֣שׁ וְאֵ֣ין כֹּ֑למִ֝תְרוֹשֵׁ֗שׁ וְה֣וֹן רָֽב׃
 **(ח)** כֹּ֣פֶר נֶפֶשׁ־אִ֣ישׁ עׇשְׁר֑וֹוְ֝רָ֗שׁ לֹא־שָׁמַ֥ע גְּעָרָֽה׃
 **(ט)** אוֹר־צַדִּיקִ֥ים יִשְׂמָ֑חוְנֵ֖ר רְשָׁעִ֣ים יִדְעָֽךְ׃
 **(י)** יִתֵּ֣ן מַצָּ֑הוְאֶת־נ֖וֹעָצִ֣ים חׇכְמָֽה׃
 **(יא)** ה֭וֹן מֵהֶ֣בֶל יִמְעָ֑טוְקֹבֵ֖ץ עַל־יָ֣ד יַרְבֶּֽה׃
 **(יב)** תּוֹחֶ֣לֶת מַחֲלָה־לֵ֑בוְעֵ֥ץ חַ֝יִּ֗ים תַּאֲוָ֥ה בָאָֽה׃
 **(יג)** בָּ֣ז יֵחָ֣בֶל ל֑וֹוִירֵ֥א מִ֝צְוָ֗ה ה֣וּא יְשֻׁלָּֽם׃
 **(יד)** תּוֹרַ֣ת מְק֣וֹר חַיִּ֑יםלָ֝ס֗וּר מִמֹּ֥קְשֵׁי מָֽוֶת׃
 **(טו)** שֵֽׂכֶל־ט֭וֹב יִתֶּן־חֵ֑ןוְדֶ֖רֶךְ בֹּגְדִ֣ים אֵיתָֽן׃
 **(טז)** יַעֲשֶׂ֣ה בְדָ֑עַתוּ֝כְסִ֗יל יִפְרֹ֥שׂ אִוֶּֽלֶת׃
 **(יז)** מַלְאָ֣ךְ יִפֹּ֣ל בְּרָ֑עוְצִ֖יר אֱמוּנִ֣ים מַרְפֵּֽא׃
 **(יח)** רֵ֣ישׁ פּוֹרֵ֣עַ מוּסָ֑רוְשֹׁמֵ֖ר תּוֹכַ֣חַת יְכֻבָּֽד׃
 **(יט)** תַּאֲוָ֣ה תֶּעֱרַ֣ב לְנָ֑פֶשׁוְתוֹעֲבַ֥ת כְּ֝סִילִ֗ים ס֣וּר מֵרָֽע׃
 **(כ)** אֶת־חֲכָמִ֣ים וְרֹעֶ֖ה כְסִילִ֣ים יֵרֽוֹעַ׃
 **(כא)** תְּרַדֵּ֣ף רָעָ֑הוְאֶת־צַ֝דִּיקִ֗ים יְשַׁלֶּם־טֽוֹב׃
 **(כב)** ט֗וֹב יַנְחִ֥יל בְּנֵֽי־בָנִ֑יםוְצָפ֥וּן לַ֝צַּדִּ֗יק חֵ֣יל חוֹטֵֽא׃
 **(כג)** רׇב־אֹ֭כֶל נִ֣יר רָאשִׁ֑יםוְיֵ֥שׁ נִ֝סְפֶּ֗ה בְּלֹ֣א מִשְׁפָּֽט׃
 **(כד)** חוֹשֵׂ֣ךְ שׂוֹנֵ֣א בְנ֑וֹוְ֝אֹהֲב֗וֹ שִׁחֲר֥וֹ מוּסָֽר׃
 **(כה)** צַדִּ֗יק לְשֹׂ֣בַע נַפְשׁ֑וֹוּבֶ֖טֶן רְשָׁעִ֣ים תֶּחְסָֽר׃
