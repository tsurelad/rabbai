# תהלים פרק קג (עם טעמים)

**(א)** לְדָוִ֨ד בָּרְﬞכִ֣י אֶת־יְהֹוָ֑הוְכׇל־קְ֝רָבַ֗י אֶת־שֵׁ֥ם קׇדְשֽׁוֹ׃
 **(ב)** בָּרְﬞכִ֣י אֶת־יְהֹוָ֑הוְאַל־תִּ֝שְׁכְּחִ֗י כׇּל־גְּמוּלָֽיו׃
 **(ג)** הַסֹּלֵ֥חַ לְכׇל־עֲוֺנֵ֑כִיהָ֝רֹפֵ֗א לְכׇל־תַּחֲלוּאָֽיְכִי׃
 **(ד)** הַגּוֹאֵ֣ל מִשַּׁ֣חַת חַיָּ֑יְכִיהַֽ֝מְעַטְּרֵ֗כִי חֶ֣סֶד וְרַחֲמִֽים׃
 **(ה)** הַמַּשְׂבִּ֣יעַ בַּטּ֣וֹב עֶדְיֵ֑ךְתִּתְחַדֵּ֖שׁ כַּנֶּ֣שֶׁר נְעוּרָֽיְכִי׃
 **(ו)** עֹשֵׂ֣ה צְדָק֣וֹת יְהֹוָ֑הוּ֝מִשְׁפָּטִ֗ים לְכׇל־עֲשׁוּקִֽים׃
 **(ז)** יוֹדִ֣יעַ דְּרָכָ֣יו לְמֹשֶׁ֑הלִבְנֵ֥י יִ֝שְׂרָאֵ֗ל עֲלִילוֹתָֽיו׃
 **(ח)** רַח֣וּם וְחַנּ֣וּן יְהֹוָ֑האֶ֖רֶךְ אַפַּ֣יִם וְרַב־חָֽסֶד׃
 **(ט)** לֹא־לָנֶ֥צַח יָרִ֑יבוְלֹ֖א לְעוֹלָ֣ם יִטּֽוֹר׃
 **(י)** לֹ֣א עָ֣שָׂה לָ֑נוּוְלֹ֥א כַ֝עֲוֺנֹתֵ֗ינוּ גָּמַ֥ל עָלֵֽינוּ׃
 **(יא)** כִּ֤י כִגְבֹ֣הַּ עַל־הָאָ֑רֶץגָּבַ֥ר חַ֝סְדּ֗וֹ עַל־יְרֵאָֽיו׃
 **(יב)** כִּרְחֹ֣ק מִֽמַּעֲרָ֑בהִֽרְחִ֥יק מִ֝מֶּ֗נּוּ אֶת־פְּשָׁעֵֽינוּ׃
 **(יג)** כְּרַחֵ֣ם אָ֭ב עַל־בָּנִ֑יםרִחַ֥ם יְ֝הֹוָ֗ה עַל־יְרֵאָֽיו׃
 **(יד)** כִּי־ה֭וּא יָדַ֣ע יִצְרֵ֑נוּזָ֝כ֗וּר כִּי־עָפָ֥ר אֲנָֽחְנוּ׃
 **(טו)** כֶּחָצִ֣יר יָמָ֑יוכְּצִ֥יץ הַ֝שָּׂדֶ֗ה כֵּ֣ן יָצִֽיץ׃
 **(טז)** כִּ֤י ר֣וּחַ עָֽבְרָה־בּ֣וֹ וְאֵינֶ֑נּוּוְלֹֽא־יַכִּירֶ֖נּוּ ע֣וֹד מְקוֹמֽוֹ׃
 **(יז)** וְחֶ֤סֶד יְהֹוָ֨ה מֵעוֹלָ֣ם עַל־יְרֵאָ֑יווְ֝צִדְקָת֗וֹלִבְנֵ֥י בָנִֽים׃
 **(יח)** לְשֹׁמְרֵ֥י בְרִית֑וֹוּלְזֹכְרֵ֥י פִ֝קֻּדָ֗יו לַעֲשׂוֹתָֽם׃
 **(יט)** יְֽהֹוָ֗ה הֵכִ֣ין כִּסְא֑וֹוּ֝מַלְכוּת֗וֹ בַּכֹּ֥ל מָשָֽׁלָה׃
 **(כ)** בָּרְﬞכ֥וּ יְהֹוָ֗המַלְאָ֫כָ֥יוגִּבֹּ֣רֵי כֹ֭חַ עֹשֵׂ֣י דְבָר֑וֹלִ֝שְׁמֹ֗עַ בְּק֣וֹל דְּבָרֽוֹ׃
 **(כא)** בָּרְﬞכ֣וּ כׇּל־צְבָאָ֑יומְ֝שָׁרְתָ֗יו עֹשֵׂ֥י רְצוֹנֽוֹ׃
 **(כב)** בָּרְﬞכ֤וּ יְהֹוָ֨ה כׇּֽל־מַעֲשָׂ֗יובְּכׇל־מְקֹמ֥וֹת מֶמְשַׁלְתּ֑וֹבָּרְﬞכִ֥י נַ֝פְשִׁ֗יאֶת־יְהֹוָֽה׃
